# FitPulse - 3D Fitness Tracker

FitPulse is a modern web-based fitness tracking application designed to help users monitor their daily activities and stay motivated to achieve their health goals.

## Features

- User login and personalized dashboard
- Track steps, calories burned, distance, and activity time
- Interactive progress rings and animated statistics
- Workout timer for activities like running, cycling, yoga, and weight training
- Weekly activity bar chart and detailed statistics
- Goal setting for steps, calories, distance, and water intake
- Notifications and reminders for fitness activities
- Profile management with customizable avatar
- Appearance settings including theme color and font size
- Multi-language support (English, Spanish, Hindi, Urdu)
- Data saved locally using LocalStorage

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage for data persistence
- SVG animations and interactive UI components

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

No installation or backend server is required.
